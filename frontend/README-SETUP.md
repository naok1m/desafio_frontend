# Frontend - Como executar

## Requisitos
- Node.js 18+ (recomendado)
- npm

## Instalação
No terminal, entre na pasta `frontend`:

```bash
cd frontend
npm install ou npm i
```

## Executar em desenvolvimento
```bash
npm run dev
```

Depois, abra no navegador a URL mostrada no terminal (normalmente `http://localhost:5173`).
